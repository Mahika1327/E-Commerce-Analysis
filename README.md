# E-Commerce-Analysis
Built an interactive E-Commerce Analytics Dashboard using Streamlit, turning raw data into actionable insights. Features include revenue, orders, profit, ratings, filters, and visuals like heatmaps &amp; trends. Strengthened skills in EDA, preprocessing, and UI design. Still learning :)

# app2.py
Consists of entire code of my E-commerce Analytics Project. I have first imported libraries like numpy, pandas, plotly etc. and with the help of CSS gamified the entire project built on layout. Apart from that, the code to visuals are specified with their respective comments for programmer's knowledge.

# ecommerce.ipynb
This consists of the code to import the dataset into JupyterNotebook and 'listdir()' function to check for the files present in main directory of JupyterNotebook. 

# python_dataset.xlsx
It is the source of dataset to my project, downloaded from publicly available datasets like kaggle, and manually merged and preproccessed to fit the needs of my project.
